### Exercice 4 
Réaliser une classe utilitaire concernant des tableaux de tableaux de double contenant les méthodes statiques suivantes :
affiche(double t[][]) : affiche les valeurs de t, à raison d’une ligne d’écran pour une ligne du tableau.
boolean regulier(double t[][]) : teste si le tableau t est régulier (toutes les lignes ont la même taille).
double[] sommeLignes(double t[][]) : renvoie un tableau contenant les sommes des différentes lignes de t.
double[][] somme(double[][] t1, double[][] t2) : vérifie que t1 et t2 sont réguliers et de mêmes dimensions et renvoie leur somme ; sinon renvoie null.
Écrire un petit programme de test.
